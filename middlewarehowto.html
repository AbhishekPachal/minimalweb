Minimalweb

A simple web framework with node.js which has provision to support rest api and http request both

Middlewares: How to?

If we want to use custom middleware to manipulate the request and response, we will create the middleware as -

File - firstMiddleWare.js

// Access the Application Logger
var logger = require('minimalweb').logger;

// The Middleware to create (Here is a dummy implementation)
var firstFunction = function (req,res,next) {
      logger.log("first Function executed");
      logger.log(req.url);
      next();
  };

module.exports = function(req,res,next) {
      return firstFunction(req,res,next);
  };

Now to put the above middleware with the framework, we need to attach this one before server starting point as -

//code snippet to enable middleware functionality

// Need for routing functionality
var route = minimalweb.route;

//Assuming the middleware in 'example-middlewares' folder
var fMiddle = require('./example-middlewares/firstMiddleWare');
// Attach the middleware
route.use(fMiddle);
